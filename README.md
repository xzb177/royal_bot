👑 Royal Bot (皇家会所)
Royal Bot 是一个专为 Emby 媒体服务器 社区打造的 Telegram 机器人。它不仅仅是一个简单的查询工具，更是一个集成了账号绑定、经济系统、卡牌收集 (Gacha) 和 社区互动 的全能管家。
旨在通过趣味性的互动（如每日签到、抽卡、排行榜），提高 Emby 频道的用户粘性和活跃度。
✨ 主要功能
• 📺 Emby 深度集成
• 水晶球占卜 (/status)：实时查看服务器负载、在线人数及正在播放的影片。
• 契约系统 (/bind)：支持用户将 Telegram 账号与 Emby 账号绑定，实现自动化管理。
• 许愿池 (/request)：用户可直接在群内提交求片需求，管理员后台可见。
• 💰 趣味经济系统
• 每日祈福 (/daily)：签到系统，包含随机运势（大吉/大凶）和连签奖励。
• 魔力转赠 (/gift)：群友之间可以互相转账心愿值。
• 魔法银行 (/loan)：提供基础的救济金机制。
• 🃏 命运卡牌 (Gacha)
• 观星抽卡 (/poster)：消耗心愿值抽取 Emby 库内的电影海报。
• 收藏系统：包含 N/R/SR/SSR 稀有度分级，打造专属的影视收藏墙。
• 🛡️ 智能管理
• 阅后即焚：自动清理刷屏消息，保持群聊整洁。
• 入群欢迎：新成员入群自动发送引导卡片。
• 数据备份：支持管理员一键备份数据库，确保数据安全。
🛠️ 技术栈
• 语言: Python 3.12+
• 框架: python-telegram-bot (Async)
• 数据库: SQLite3
• 部署: Systemd Service / Docker (Planned)
